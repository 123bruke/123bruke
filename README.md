# 🏥 Linear Regression Health Cost Calculator

This project predicts healthcare insurance costs using **TensorFlow Linear Regression**.  
It analyzes different personal factors and estimates medical expenses using a machine learning model.

The project was created as part of the **freeCodeCamp Machine Learning with Python Certification**, which focuses on applying machine learning techniques to real-world problems.

## 🚀 Features
- Predicts healthcare insurance costs
- Built using **TensorFlow regression**
- Demonstrates **supervised machine learning**
- Uses real-world healthcare cost dataset

## 🧠 Technologies Used
- Python
- TensorFlow
- NumPy
- Pandas
- Matplotlib

## 📚 Project Purpose
The goal of this project is to understand how **regression models** work and how machine learning can be used to predict continuous values such as healthcare costs.
